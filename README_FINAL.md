# Wonder Words - Word Guessing Game

A fun interactive word-guessing game built with PHP, MySQL, and JavaScript. Test your vocabulary across three difficulty levels!

## Features

- **3 Difficulty Categories**: Easy, Moderate, and Difficult
- **User Authentication**: Login and registration system
- **Scoring System**: Earn points for correct answers
- **Clue System**: Use strategic clues to help guess words
- **Leaderboard**: Compete with other players
- **Audio Effects**: Immersive sound effects throughout the game

## Gameplay

- **Earn 10 points** for each correct word
- **3 strikes** before losing the game
- **3 clues available** (25 points each) - choose between vowel or consonant hints
- **Complete all 3 categories** to win the game

## Tech Stack

- **Backend**: PHP 8.2
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript
- **Audio**: Custom sound effects

## How to Play

1. Register or login to your account
2. Click "Play" to start the game
3. Guess letters to reveal the hidden word
4. Use clues strategically to help solve harder words
5. Earn points by completing words
6. Try to beat the leaderboard!
